
# i3wm-rice

This rice is mainly coded for Arch, but can be used by any distrobution with slight tweaking. 


## Information
Operating System: Arch Linux
Window Manager: i3
dmenu launcher: rofi
Audio manager: pipewire-pulse

## Dependencies

- i3wm
- picom
- feh
- polybar
- kitty terminal
- neovim
- rofi
- betterlockscreen
